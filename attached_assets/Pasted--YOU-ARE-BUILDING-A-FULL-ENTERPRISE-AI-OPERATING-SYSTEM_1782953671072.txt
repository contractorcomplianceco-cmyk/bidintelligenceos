🧠 CORE SYSTEM — BidIntelligenceOS
────────────────────────────────────

BidIntelligenceOS is the CENTRAL INTELLIGENCE ENGINE.

It manages full lifecycle:

Opportunity → Intelligence → Bid → Win/Loss → Job Execution → Profit Tracking → Learning Loop

CORE FUNCTIONS:
- bid intake
- scope intelligence extraction
- risk analysis engine
- bid generation
- proposal creation
- job auto-creation after win
- scheduling engine
- permit tracking system
- cost tracking (labor, subs, materials)
- ROI tracking
- change order detection
- execution monitoring
- performance feedback loop (Bid DNA learning)

WHEN BID IS CREATED:
1. System analyzes scope
2. Detects missing info
3. Assigns risk score
4. Generates bid structure
5. Builds client-ready proposal

WHEN BID IS WON:
1. Job is auto-created
2. Subcontractors assigned via BuildConnect
3. Schedule auto-generated
4. Permits tracking activated
5. Cost tracking starts
6. Weather monitoring activated

WHEN JOB IS ACTIVE:
- real-time cost monitoring
- labor tracking
- sub performance scoring
- margin drift detection
- delay detection
- risk alerts

WHEN JOB IS COMPLETE:
- profit vs estimate analysis
- performance scoring
- bid DNA learning updates
